⚡ Quick Start
  1. Set GROQ_API_KEY and GEMINI_API_KEY in your .env file
  2. Start Qdrant: docker-compose up qdrant -d
  3. Go to Generate → paste a Google Drive folder ID → click Sync & Ingest
  4. Once ingestion completes, select the client → choose content type → click Generate Content
  5. Click Generate Brand Voice Doc to create a brand guide and upload it back to Drive
